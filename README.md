# PFD / P&ID Drawer

A self-contained, browser-based **Process Flow Diagram editor** for water-treatment plant
schematics. Single HTML file, no backend, works offline. Drag equipment onto the canvas,
connect data-carrying pipes, auto-generate line & equipment schedules, and export SVG/PNG.
Ships with UF / RO / UF+RO process templates.

**Live app:** https://adrianyeo77-coder.github.io/pnid-drawer/
